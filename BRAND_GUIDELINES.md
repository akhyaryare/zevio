# Zevio — Brand Voice Guidelines

> Version 1.0 — April 2026

---

## Executive Summary

Zevio's voice is Akhyar's voice. Direct, personal, empathetic, and completely free of corporate language. The brand speaks as one person who genuinely understands the pain of running a UK local business.

**North star sentence:** "You do not need to work faster. You need the work gone."

---

## We Are / We Are Not

| We Are | We Are Not |
|---|---|
| Personal — Akhyar's name, number, and email visible everywhere | Faceless — not a platform, not "our team" |
| Quantified — every claim has a number | Vague — "improve your business" is not our language |
| Plain English — written for restaurant owners and barbers, not CTOs | Jargon-heavy — no "AI-powered workflows" or "seamless integrations" |
| Problem-first — name what's costing time before mentioning the product | Feature-first — we don't lead with WhatsApp automation or Stripe |
| Risk-free by design — 30-day money-back, no contracts | Predatory — no lock-in, no hidden fees, no pressure |
| Reframing — "You don't need to work faster. You need the work gone." | Incremental — we don't promise marginal improvement, we promise removal |
| UK-rooted — GBP pricing, UK cities, UK phone number | Generic global — not a one-size-fits-all international tool |
| Before/after — everything shown as a transformation | Status quo comfortable — always show what's being replaced |

## Voice Attributes

### Direct & Unfiltered
Say the thing plainly. Short declarative sentences. Facts before feelings.
- YES: "I remove hours of daily admin from your business."
- NO: "We're passionate about helping businesses succeed."

### Pragmatic Problem-Solver
Every feature is described by what it eliminates.
- YES: "Simple systems that handle your daily admin automatically. No complicated software. No staff needed."
- NO: "Revolutionise your operations with cutting-edge AI."

### Empathetic (Not Sympathetic)
Validate the problem without blame. They're not failing — they're flat out.
- YES: "Not because they didn't care, but because they were already flat out."
- NO: "We know how overwhelming it can be running a business."

### Results-Obsessed
Every claim is quantified. Every feature anchored to a measurable result.
- YES: "Most businesses save 10-15 hours per week."
- NO: "Better customer experience" / "improve responsiveness"

### Founder-Led Personal Brand
Akhyar is not hiding behind the product. First person always.
- YES: "Hi, I'm Akhyar. I build automation systems for UK restaurants, barbers, and salons."
- NO: "Our team of experts", "the Zevio platform", "our proprietary technology"

### Anti-Corporate / Transparent
Conspicuously avoids anything that sounds like a faceless company.
- YES: "No long contracts, cancel anytime." / "Full refund. No questions asked."
- NO: "Contact our sales team" / "Competitive pricing"

## Terminology Guide

### Must-Use Terms

| Term | Instead Of |
|---|---|
| Admin | Workflows, processes, tasks |
| Systems | Software, platform, tool |
| Runs automatically | Enables, helps manage, automates the process of |
| Instant | Fast, quick, rapid |
| Handles | Manages, assists with, supports |
| Saves / Saved (+ number) | Reduces, improves, optimises |
| Gone | Reduced, minimised, addressed |
| Set up | Implemented, deployed, configured |

### Never Use

| Term | Reason |
|---|---|
| Revolutionary / innovative | Undermines trust |
| Passionate about / excited to | Corporate filler |
| Our team | Contradicts founder-led positioning |
| World-class / best-in-class | Empty superlatives |
| Workflows / processes | Too corporate |
| Streamline / optimise | Generic, meaningless |
| AI-powered / cutting-edge | Tech-bro language |
| Solution / platform | Corporate SaaS register |
| Onboard / deploy | Tech jargon |
| Limited time offer / Act now | Manipulative; contradicts no-pressure stance |

## Key Phrases

- "You do not need to work faster. You need the work gone."
- "[X hours / X pounds] -- gone every morning."
- "Every week without automation is a week of silent losses."
- "Zevio costs less than one dinner service per month."

- "From day one, your system handles..."

## Objection Handling

**"I can't afford it."**
"Missing just one £60 table booking per week covers it. The question isn't whether you can afford it -- it's how much longer you can afford not to."

**"I'm not very technical."**
"You don't touch any software. I set everything up for you. You just get your time back."

**"What if it doesn't work?"**
"If it doesn't save you time in 30 days, full refund. No questions asked."

**"Why not use Zapier/Make?"**
"Zapier requires you to build it yourself. I build it for you. You don't touch any software."

## Tone by Context

| Context | Tone |
|---|---|
| Landing page hero | Inspiring, punchy -- name the pain then reframe |
| Problem section | Validate, don't dramatise |
| Feature descriptions | Benefit first, mechanism never |
| Pricing | Anchor cost to familiar spend; remove risk |
| Founder bio | Personal, honest, mission-driven |
| CTA | Direct, single action, no pressure |
| Cold outreach | Open with a specific cost they're paying |
